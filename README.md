## Usage
1. Open the demo at <a href="https://sailos.netlify.app" target="blank">sailos.netlify.app<a>
2. To open an app just press one of the icons

<bold>Weather app<bold>
1. Input the place
2. Look at the results

<bold>Map app<bold>
1. input the place 
2. use your mouse to drag the map
3. press the draw button to draw
4. to draw left click on the map to place a point and press a second time to draw a line between
5. press the remove button to remove all the drawings

## features
- working map with draw function
- accurate weather prediction
- working dragging and expanding
